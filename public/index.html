Key features of mistX

  • 1. Flashbots!

    The unique technology underlying the platform uses Flashbots and does not publish user transactions to a public mempool. Frontrunners be gone.

  • 2. Failed swap? No fees

    There is no concept of reverts on mistX. Your transaction is either successful or you pay nothing.

  • 3. Self - sustaining

    A proportion of trade value on mistX will be reserved to enrich future Alchemist rewards programs.

  • 3. Rewards

    Users can receive cashback on applicable trades. Find out more.

Flashbots as a Service


The Flashbots Protect API provides a simple way for developers to integrate Flashbots into their dApps. This is an important piece of infrastructure that will enable the next generation of Flashbots users to be onboarded and help minimize the negative effects of MEV in Ethereum.



The first exciting product for end users we built to use this API is the Flashbots RPC. Users will now be able to send transactions to Flashbots directly from their wallet such as Metamask. Adding this custom RPC to your wallet means you can use Flashbots technology with you where ever you go. It can be to send coins, swap on any DEX or mint NFTs - all using the Flashbots Network.
